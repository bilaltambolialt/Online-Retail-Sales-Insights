# Online Retail Sales Data (Refined)

## Overview

This repository contains a refined version of an Online Retail Sales dataset. The data has been cleaned and prepared for business analysis, reporting, and data visualization.

## Dataset Description

The refined dataset includes transactional records from an online retail business. It is suitable for analyzing customer purchasing behavior, sales performance, product trends, and revenue insights.

## Data Cleaning Performed

The following preprocessing steps were applied:

* Removed duplicate records.
* Handled missing or null values.
* Corrected inconsistent data formats.
* Standardized column names.
* Converted date fields into a consistent datetime format.
* Removed invalid or negative quantity/value records where applicable.
* Trimmed unnecessary spaces and corrected text formatting.
* Verified data types for numerical and categorical columns.
